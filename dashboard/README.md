# Revenue Leakage Dashboard

This folder contains the **Revenue Leakage Analysis Dashboard** for the e-commerce revenue study.

## Contents

- `Revenue Leakage Analysis Dashboard.pdf` – Full multi-page dashboard showing insights on revenue leakage across customer behavior, delivery performance, and operational processes.  
- `dashboard_preview.png` – Optional preview of the first page for quick reference.  
- `Revenue_Leakage_Dashboard.pbix` – Power BI source file (not included due to large file size).

## How to View

1. Click the PDF file to open it directly in GitHub and scroll through the pages.  
2. Use the preview image to get a quick idea of the dashboard layout.
